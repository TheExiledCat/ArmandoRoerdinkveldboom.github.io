Monument Valley Remake

Isometric 3d illusion based puzzle game remade for pc as a vertical slice

About Monument Valley

This is a game i made with 3 artists for a school assignment called Vertical Slice. In this assignment we were tasked of recreating 10 seconds of game mechanics from a game from scratch. We had to make all the art and code for the first level of the game in this case. The game makes u take control of a cute little character that has to beat puzzles in an isometric perspective. Becus of this perspective you can do some nifty things like walk over to the forground and background elements as if they are one. On this project i worked as the teamleader and as the sole game programmer

Programs and languages used

  • Unity Engine
  • Unity C#
  • Photoshop
  • AutoDesk Maya
  • Adobe Illustrator
  • Audacity

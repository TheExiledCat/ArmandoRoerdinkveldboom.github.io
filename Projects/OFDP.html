About One finger death punch

This is a project i made for the second part of the vertical slice 2019 for my school assignment. This project like my monument valley vertical slice used 4 artists and myself. we made the game based on a 10 second clip we got to choose ourself unlike the monument valley game that was a predetermined choice. We made the game in unity and had a ton of fun doing it since it was a game we all liked ourself and we also loved oldschool chinese martial art films.

Programs and languages used

  • Unity Engine
  • Unity C#
  • Phostoshop
  • Unity Engine Animator
  • Audacity
